# 🎓 SkillBridge AI

SkillBridge AI is an AI-powered career guidance platform built using Flask and Google Gemini AI. It helps students improve their resumes, identify skill gaps, prepare for interviews, and generate personalized career roadmaps.

---

## 🚀 Features

- 📄 AI Resume Analyzer
- 🤖 AI Career Mentor
- 🛣️ Personalized Career Roadmap Generator
- 🎤 AI Mock Interview Questions
- 📊 Skill Gap Analyzer
- 📚 Previous Resume Reports
- 🔐 Login System
- 💾 SQLite Database
- ⚡ Google Gemini AI Integration
